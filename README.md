# 🏦 Ryt Bank - Secure Transaction History Module

A React Native application built for the Frontend Engineer take-home assessment. This module securely displays a user's transaction history, utilizing device biometrics (FaceID/Fingerprint) to mask and reveal sensitive transaction amounts.

## 🚀 Quick Start (Recommended)

To save time setting up a local environment, you can download the pre-compiled Android build to test on your emulator or physical device.

*   🤖 **[Download Android APK Here](https://expo.dev/artifacts/eas/nQF7qQDRh9EgvAXH1pLNtQ.apk)**
    *   *How to use:* Simply drag and drop the downloaded `.apk` file into an active Android Studio Emulator, or transfer it to a physical Android device to install.

*(Note: iOS build is omitted as this was compiled on a non-macOS environment, but the codebase is fully cross-platform via Expo).*
